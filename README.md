# net-chat
Простой сетевой чат для ПК с авторизацией на сервере.

Написан на Java с использованием JavaFX.
Приложение позволяет обмениваться сообщениями в общем чате, писать личные сообщения.
Есть возможность регистрации нового пользователя с дальнейшей авторизацией.
